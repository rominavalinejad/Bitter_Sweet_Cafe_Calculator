# Cafe Bitter Sweet - Bill & Share Calculator

## About the Project

On my second day of continuous Python learning, right after mastering Data Types and Variables, I developed this simple yet practical script to solidify my understanding. 

This program calculates the total bill and split shares for a real-world hangout at our favorite spot in Tehran, Bitter Sweet Cafe, using actual prices from their online menu. While it assumes an equal split for simplicity, I also simulated a dynamic, real-life scenario: *the sudden arrival of a new friend!* 

As a result, the script recalculates and displays the updated total cost and per-person share in two distinct stages (4-person vs. 5-person).
